# Predictive Modeling Using Machine Learning

## Project Overview
This project demonstrates a supervised machine learning workflow to predict customer churn using the Random Forest Classifier. It includes data preprocessing, model training, prediction, and performance evaluation.

## Objective
Build a machine learning model to predict whether a customer is likely to churn based on customer attributes.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- OpenPyXL

## Machine Learning Algorithm
- Random Forest Classifier

## Workflow
1. Load the dataset
2. Handle missing values
3. Remove duplicate records
4. Clean and encode categorical data
5. Split the dataset into training and testing sets
6. Train the Random Forest model
7. Make predictions
8. Evaluate the model using Accuracy Score
9. Visualize results with a Confusion Matrix and ROC Curve

## Files Included
- customer_churn_raw.xlsx
- customer_churn_cleaned.xlsx
- Predictive_Modeling.ipynb
- requirements.txt

## Outcome
Successfully built and evaluated a predictive machine learning model using supervised learning techniques.
